Core Components

  • MusicGame: Controls everything, manages the game and player actions, handles game flow and progression, and adapts with the score.
  • Background: Sets the scene, handles collisions, adjusts visuals and gameplay elements, and resets for new challenges.
  • Player: The player's character, moves, scores points, and reacts to collisions, keeping track of progress and special bonuses.
  • Paddle: Helps control the game by moving and adjusting its size, changing speed and size as you progress.
  • PowerUp: Provides special boosts during the game, appears randomly, and alters gameplay momentarily.
  • Sound: Adds music and effects to enhance the game atmosphere, reacts to game events to keep the player engaged.
  • MainMenu: Starts your adventure, offering options to begin, pause, and restart the game.

How to Play

To play Music Game Adventure, follow these steps:

  1. Visit the GitHub repository: https://github.com/RFahal01/Music-Game
  2. Click on the green "Code" button and select "Download ZIP" to download the game files.
  3. Extract the downloaded ZIP file to your desired location on your computer.
  4. Open your terminal or command prompt and navigate to the extracted folder.
  5. Ensure you have Java installed on your computer. If not, download and install it from the official Java SE Downloads page.
  6. Compile the game by running the following command in your terminal: 
    javac MusicGame.java
  7. Run the game by executing the following command: 
    java MusicGame
  8. Use the arrow keys to navigate and the spacebar to jump. Collect musical notes and avoid obstacles to earn points and power-ups.
